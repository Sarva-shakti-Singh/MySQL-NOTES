# 📊 SQL Notes Repository

## 🚀 Overview

Welcome to the **SQL Notes Repository** – a complete, structured, and easy-to-understand guide to mastering SQL from basics to advanced concepts.

This repo is designed especially for:

* 🎓 B.Tech / CS Students
* 💻 Beginners learning databases
* 🧠 Interview & exam preparation

---

## 📚 What You'll Learn

### 🔹 Basics

* Introduction to SQL
* DDL, DML, DCL, TCL Commands
* Data Types & Constraints

### 🔹 Intermediate

* Joins (INNER, LEFT, RIGHT, FULL)
* Subqueries & Nested Queries
* Aggregate Functions

### 🔹 Advanced

* Indexing & Optimization
* Transactions & ACID Properties
* Views, Stored Procedures, Triggers

### 🔹 Database Design

* ER Models
* Normalization (1NF → BCNF)
